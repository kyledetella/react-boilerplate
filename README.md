React Boilerplate
=================

Everything you need to get up and running with React quickly!

```bash
$ npm i && npm start
```

This will run the app on port `3045`.

http://localhost:3045

## Test

```bash
$ npm t
```

Watch tests

```bash
$ npm run test:watch
```
